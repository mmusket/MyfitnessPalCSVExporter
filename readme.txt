This repository contains the code for the chrome extension 
https://chrome.google.com/webstore/detail/ejigkdnkmegmdnemjpnfmnapmpgjeopn?hl=en&gl=GB



To use the extension follow these steps:
-Go to Food page
-Scroll down 
-Click on "View all report" 
-Select the range of dates you want to export 
-Click on the export CSV button 

the file downloaded has no type, but if you rename it to something.csv it should open in excel
